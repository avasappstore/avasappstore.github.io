    AVAS

    What is AVAS?

    Avas is short for Alex Vegas App Store. This Store contains a collection of well designed and powerful apps for the android paltform. All these apps are open source. They have all been tested for security and functiomality and passed both test with flying colours.

    How do I install an APK?

    Download the APK from a safe source. Then, go to Settings>Security and make sure that "Unknown Sources" is checked. After that, head over to a file amanger and navigate to the folder that contains the APK and tap on the file. Android will ask you wheether you want to install this app. Click on"yes" and install the file. You're all set! Congrats!
